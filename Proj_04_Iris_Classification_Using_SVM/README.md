# Iris Classification using Support Vector Machine (SVM)

## Objective
Build a Support Vector Machine (SVM) classifier to distinguish between two Iris flower species using sepal length and sepal width.

## Technologies Used
- Python
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Iris Dataset (built-in from Scikit-learn)
- **Input Features:**
  - Sepal Length
  - Sepal Width
- **Target Variable:**
  - Iris Species (First two classes)

## Workflow
1. Load the Iris dataset.
2. Select the first 100 samples and two input features.
3. Train a Linear Support Vector Machine (SVM) classifier.
4. Predict the flower species.
5. Evaluate the model using a Confusion Matrix and Accuracy Score.
6. Visualize the data along with the SVM decision boundary.

## Results / Output
The program displays:
- Scatter plot of the Iris dataset
- SVM decision boundary
- Confusion Matrix
- Classification Accuracy Score
