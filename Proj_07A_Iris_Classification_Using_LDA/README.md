# Iris Classification using Linear Discriminant Analysis (LDA)

## Objective
Build a Linear Discriminant Analysis (LDA) classifier to classify Iris flower species using the Iris dataset.

## Technologies Used
- Python
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Iris Dataset (built-in from Scikit-learn)
- **Input Features:**
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- **Target Variable:**
  - Iris Species (First two classes)

## Workflow
1. Load the Iris dataset.
2. Select the first 100 samples.
3. Split the dataset into training and testing sets.
4. Train a Linear Discriminant Analysis (LDA) classifier.
5. Predict the flower species.
6. Evaluate the model using a Confusion Matrix and Accuracy Score.
7. Visualize the original data and the predicted classifications.

## Results / Output
The program displays:
- Scatter plot of the original Iris dataset
- Scatter plot of the predicted classifications
- Confusion Matrix
- Classification Accuracy Score
