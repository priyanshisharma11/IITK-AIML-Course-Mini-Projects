# Car Purchase Prediction using Decision Tree

## Objective
Build a Decision Tree classifier to predict whether a customer is likely to purchase a new car based on age and estimated salary.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Purchase_Logistic.csv
- **Input Features:**
  - Age
  - Estimated Salary
- **Target Variable:**
  - Purchased (1 = Yes, 0 = No)

## Workflow
1. Load the customer purchase dataset.
2. Select age and estimated salary as input features.
3. Split the dataset into training and testing sets.
4. Train a Decision Tree classifier.
5. Predict customer purchase decisions.
6. Evaluate the model using a Confusion Matrix and Accuracy Score.
7. Visualize the Decision Tree.
8. Display the text representation of the Decision Tree.

## Results / Output
The program displays:
- Decision Tree visualization
- Text representation of the Decision Tree
- Confusion Matrix
- Classification Accuracy Score
