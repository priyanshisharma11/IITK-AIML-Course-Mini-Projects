# Linear Regression on Iris Dataset

## Objective
Predict sepal width using sepal length from the Iris dataset with a Linear Regression model.

## Technologies Used
- Python
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Iris Dataset (built-in from Scikit-learn)
- **Input Feature:** Sepal Length
- **Target Variable:** Sepal Width

## Workflow
1. Load the Iris dataset.
2. Select the first 50 samples.
3. Train a Linear Regression model.
4. Predict sepal width values.
5. Evaluate the model using MSE and R² Score.
6. Visualize Actual vs Predicted values.

## Results / Output
The program displays:
- Training Mean Squared Error (MSE)
- Training R² Score
- Scatter plot comparing Actual vs Predicted Sepal Width values
