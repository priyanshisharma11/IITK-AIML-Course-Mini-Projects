# Iris Clustering using K-Means

## Objective
Apply the K-Means clustering algorithm to group Iris flower samples into clusters and evaluate clustering performance using the Elbow Method and Silhouette Score.

## Technologies Used
- Python
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Iris Dataset (built-in from Scikit-learn)
- **Number of Samples:** 150
- **Number of Features:** 4
- **Number of Clusters:** 3

## Workflow
1. Load the Iris dataset.
2. Visualize the original data.
3. Apply the K-Means clustering algorithm.
4. Visualize the generated clusters.
5. Use the Elbow Method to determine the optimal number of clusters.
6. Evaluate clustering quality using the Silhouette Score.

## Results / Output
The program displays:
- Original Iris dataset visualization
- K-Means clustered visualization
- Elbow Method (SSE) plot
- Silhouette Score plot
