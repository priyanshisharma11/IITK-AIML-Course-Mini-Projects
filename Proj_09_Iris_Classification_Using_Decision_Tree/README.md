# Iris Classification using Decision Tree

## Objective
Build a Decision Tree classifier to classify Iris flower species and visualize the generated decision tree.

## Technologies Used
- Python
- Scikit-learn
- Matplotlib

## Dataset
- **Dataset:** Iris Dataset (built-in from Scikit-learn)
- **Number of Samples:** 150
- **Number of Features:** 4
- **Target Variable:** Iris Species

## Workflow
1. Load the Iris dataset.
2. Split the dataset into training and testing sets.
3. Train a Decision Tree classifier.
4. Predict the Iris species on the test dataset.
5. Evaluate the model using a Confusion Matrix and Accuracy Score.
6. Visualize the Decision Tree.
7. Display the text representation of the Decision Tree.

## Results / Output
The program displays:
- Decision Tree visualization
- Text representation of the Decision Tree
- Confusion Matrix
- Classification Accuracy Score
